Questly Question Generator
Overview

Questly is a web-based application that automatically generates questions from uploaded notes or documents. Users can upload a PDF or text file, and the system extracts content, identifies key concepts, and generates structured questions. The interface is clean, responsive, and includes features like light/dark mode, real-time question generation, and export options for easy use.

Objective

To create an automatic question generator that converts study material into structured questions.

To support PDF and text uploads with seamless text extraction.

To generate meaningful and grammatically correct questions using text processing techniques.

To provide a simple and accessible interface for learners and educators.

Tools Used

HTML, CSS, JavaScript — For front-end design and functionality

PDF.js — For text extraction from PDF files

LanguageTool API — For grammar correction and refinement

LocalStorage — For storing user preferences (like theme mode)

GitHub Pages — For deployment and hosting

Conclusion

The Questly Question Generator simplifies the process of creating educational content by converting uploaded notes into well-structured questions. With its intuitive interface, grammar correction, and smooth user experience, Questly helps students and educators save time and enhance learning efficiency.
